# dungeonslayers-d2go
Dieses Foundry-VTT-Modul liefert dir einen sofort spielbaren Einstieg in die Welt von Dungeonslayers. Es enthält Inhalte von D2go / Dungeon-2-go sowie fertige Beispielcharaktere.

#Installation
manifest: https://github.com/59zwen/dungeonslayers-d2go/releases/latest/download/module.json
Benötigt wird das Spielsystem DS4: https://foundryvtt.com/packages/ds4

#Inhalt
Bislang enthalten sind Adaptionen von:
- Spielbereite SC der Stufe 1 von https://dungeonslayers.net/download/DS4SC2Go.pdf
- Makros für die Hotbar: SC-Sheet öffnen, LK -1, LK ändern, LK +1 (Makros KI-generiert)
- D2Go 10: https://dungeonslayers.net/download/Dungeon2Go10.pdf, C. KENNIG, CC-BY-NC
- D2Go 9: https://dungeonslayers.net/download/Dungeon2Go9.pdf, C. KENNIG, CC-BY-NC
